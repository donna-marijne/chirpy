# chirpy

An HTTP CRUD server for learning purposes.

## Pre-requisites

1. PostgreSQL database (get a connection string)

## Setup

1. Copy `.env.example` to `.env`
2. Populate `DB_URL` with your PostgreSQL database connection string
3. Populate `POLKA_KEY` with your polka key :)
4. Populated `SECRET` with a random string
5. `go run .`
